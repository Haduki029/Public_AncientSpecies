## Inner class
インナークラス(内部クラス)に関しては、
・自分を所持しているクラスの内部のみで使う
・公開していたとしても、自分を所持しているクラスのメソッドの引数用
等、独立しているクラスである必要性が薄い、もしくは独立させることで混乱を招く可能性がある
そういったクラスはインナークラスにして使ったりします
尚外部に公開されてるインナークラスの場合、
OuterClass.InnerClass innerClass = new OuterClass.InnerClass();
みたいに外側のクラスから静的メンバを扱う時のようなアクセス演算子付きでのアクセスになります。
それ以外は普通のクラスと同じですにゃ。
